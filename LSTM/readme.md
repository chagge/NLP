## 動作環境

- Chainer 1.8+

## 使い方

train_textフォルダ内の`train.py`を学習に用い、`validate.py`で文章生成を行えます。

train_text内に`text`フォルダを作成し、その中に文章データの`.txt`ファイルを入れてください。

テキストファイルは何個入れてもすべて自動的に読み込まれます。ファイル名も任意で構いません。
